# Tipe Data Character

- Data Character (huruf) di Java dipresentasikan oleh tipe char
- Untuk membuat data char di Java, kita bisa menggunakan tanda ' (petik satu) di awal dan di akhir karakter

## Kode : Character
```java
char e = 'E';
char k = 'K';
char o = 'O';

System.out.print(e);
System.out.print(k);
System.out.print(o);
```
